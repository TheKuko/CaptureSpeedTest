Capture Speed Test
===
* PF_RING Vanilla, Netmap, PFQ, PcapPlusPlus

* Inspired by https://github.com/mfontanini/packet-capture-benchmarks

* Tested on Ubuntu 16.04.2 LTS

## How to start
1. Clone repository
2. cd scripts
3. ./prepare_ubuntu.sh
4. ./install_libs.sh
5. cd ../
6. make
